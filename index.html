На сайте выложены проекты для реализации. Как правило это проекты в сфере ИТ, однако тематика строго не регламентируется. Проект — описание, цели, список необходимых и уже имеющихся ресурсов для реализации проекта. Любой желающий может принять участие в создании проекта, применив свои навыки, творчество и самое главное — любовь :). По мере наполнения проекта участниками он будет менять свой статус. Цель данного ресурса — объединить творческих людей в учебном порыве для создания некоммерческих проектов.

And now, let's get down to business!

1. Roboarm

Opensource проект робоманипулятора

Разработка opensource проекта роботизированной руки. Arduino-based. Имеются все основные напечатанные на 3д принтере детали, 70% необходимой электроники. Остальное в процессе допечатывания/докупки.Проект полностью профинансирован, необходимости что-либо покупать за свой счет нет. Ссылка на оригинальный проект BCN3D-Moveo

2. Retropie

Retropie — эмулятор старых игровых приставок на Raspberry в рекреационную зону

Разработка эмулятора старых игровых приставок для общего использование в рекреационной зоне(кафедра, 3 этаж, аквариум). Имеется полный набор комплектующих (малина, джойстики, 10и дюймовый экран, коробка). Необходимо собрать это все во едино в красивой и практичной форме. Ссылка на оригинальный проект retropie

3. Робот сумоист

Робот сумоист 10х10 для победы в соревнованиях и не только.

Существует категория соревнований - робосумо youtube. Есть прошлые наробоки в этом виде соревнований. Цель - создать робота для участия в соревнованиях категории робосумо.
